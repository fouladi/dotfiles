# dotfiles

To install every configuration to your current environment use
`GNU Stow`!

## nvim

* URL: https://neovim.io
* Change to the current directory in *dotfiles*.
* Run: `stow nvim`

## wezterm - WezTerm - Wez's Terminal Emulator

* URL: https://wezfurlong.org/wezterm/index.html
* Change to the current directory in *dotfiles*.
* Run: `stow wezterm`
